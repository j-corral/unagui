### Component usage

```tsx
import { Section } from '@hoomies/unagui.components.section';
import { Text } from '@chakra-ui/react';

<Section>
  <Text>Hello</Text>
</Section>;
```

### More props

The Section component is extended from the Chakra UI Flex component.

So you can use Flex props from the official documentation.

[Chakra UI - Flex](https://chakra-ui.com/docs/layout/flex)

[CSS Flex Box Layout](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Flexible_Box_Layout)
